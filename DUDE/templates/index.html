About Us

NCSAEL's prime emphasis is to Design and Develop Indigenous Tools to carry out
Security Assessment and Auditing of foreign and local IT products/ services

Higher Education Comission (HEC) and Planning Commission of Pakistan's sponsored, National Cyber Security Auditing and Evaluation Lab is an internationally recognized Cyber-Security Research Lab established at NUST

  • Security Assessment and Auditing of foreign and local IT security products/ services
  • Indigenous Development of specialized tools/ software for carrying out security screening and malware analysis
  • Capacity Building for Common Criteria Testing Lab (CCTL) through Assessment of security targets

Product Features

Firmware Unpacking

Performs the unpacking of newly designed and unique filesystem formats

Diverse Filesystem Support

Support binaries having 49 diverse filesystem

Security Assessment

In-depth security analysis of executable binaries to detect vulnerabilities, hardcoded IPs, SSL certificates, malicious files, hidden backdoors, and so on

Cross Vendor Support

Support variety of devices i.e. routers, switches, smart watches, IP cameras from different vendors

Acquisition of Firmware Binary from Embedded Devices Firmware

FEAST makes use of flashrom, a linux utility, to acquire firmware binary using CH341A and SOIC8 flash chip IC test clip adapter

JTAG is also used over a hardware tool called Bus Blaster to connect to the JTAG interface of the embedded device target board. It then communicates with CPU and CPU in return communicates with the flash memory. In this way, contents of flash are dumped

Obtain the firmware binary from vendor websites

Security Status of Firmware Under Test

Once the filesystem unpacking and security assessment has been completed, FEAST generates a detailed report regarding vulnerabilities present in the embedded firmware devices and their associated impact level

Summary of findings provides the security status, generates the risk analysis results, type of vulnerabilities with their associated impact levels, and ways to patch the discovered vulnerabilities

Generated report covers all major findings of security assessment including, malicious code stacks, buffer overflow bugs, obsolete functions, bad argument bugs, disclosed passwords, possible hidden backdoors, and sensitive information. Furthermore, hardcoded IP address and URLs, which may be used for remote execution of malicious scripts and monitoring are present

More Features

FEAST offers the following novel features for the vulnerability analysis of firmware undertest

Encrypted Firmware Support

Successful decryption of the encrypted firmware binaries


Deobfuscation of Obfuscated Binaries

Deobfuscation of obfuscated detected magic bytes using magic bytes iterative database and superblock database

Endian Conversion of Compressed Romfs

Big endian conversion of compressed romfs to little endian format

Efficient binary extraction and analysis. Less computational complexity
